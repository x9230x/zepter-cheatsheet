# Шпаргалка Zepter — сайт с авто-обновлением цен

Одна страница-шпаргалка по 3 системам очистки воды Zepter. Хостится бесплатно
на GitHub Pages (работает 24/7, открывается на любом устройстве без входа).
Робот на GitHub Actions раз в день сам сверяет розничные цены с zepter.kz.

## Что внутри

| Файл | Зачем |
|------|-------|
| `index.html` | Сама шпаргалка. Цены подгружает из `prices.json`. |
| `prices.json` | Текущие цены и дата проверки. Его меняет робот. |
| `scripts/update-prices.mjs` | Робот: заходит на 3 страницы Zepter, берёт цены, пишет в `prices.json`. |
| `.github/workflows/update-prices.yml` | Расписание: запускает робота каждый день в 08:00 по Алматы. |

## Настройка (один раз, ~5 минут)

1. **Создай репозиторий** на GitHub: кнопка **New** → имя, например `zepter-cheatsheet`
   → **Public** → Create repository.

2. **Загрузи файлы.** На странице репозитория: **Add file → Upload files**,
   перетащи сюда всё содержимое папки `zepter-site` (включая папки `scripts`
   и `.github`). Внизу нажми **Commit changes**.

3. **Включи права робота на запись.** Settings → Actions → General →
   раздел **Workflow permissions** → выбери **Read and write permissions** → Save.

4. **Включи сайт (GitHub Pages).** Settings → Pages → Source: **Deploy from a branch**
   → Branch: **main**, папка **/(root)** → Save. Через минуту вверху появится адрес вида
   `https://ТВОЙ_ЛОГИН.github.io/zepter-cheatsheet/` — это и есть постоянная ссылка.

5. **Проверь робота.** Вкладка **Actions** → слева **Update prices** → кнопка
   **Run workflow**. Через минуту он отработает и обновит `prices.json`.
   Дальше будет запускаться сам каждый день.

Готово. Делись ссылкой из шага 4 — она работает всегда.

## Как это работает

- Открываешь ссылку → браузер грузит `index.html`, а он подтягивает свежие цены
  из `prices.json`.
- Каждый день в 08:00 (Алматы) GitHub на своих серверах запускает робота: он читает
  розничную цену с каждой страницы товара (первый атрибут `data-remarket-itemvalue`),
  и если что-то изменилось — коммитит новый `prices.json`. Твой компьютер при этом
  не нужен.
- Если сайт Zepter временно недоступен или цену не удалось прочитать — робот
  оставляет прежнее значение, а не стирает его.

## Поменять что-то вручную

- **Цены/дата** — отредактируй `prices.json` прямо на GitHub (карандаш → Commit).
- **Тексты, плюсы/минусы, возражения** — в `index.html`.
- **Время запуска** — строка `cron` в `.github/workflows/update-prices.yml`
  (время в UTC; Алматы = UTC+5, т.е. 08:00 местного = `0 3 * * *`).
